## About Catalogo Coppel

Catalogo Coppel es una aplicación web basica con el proposito de validar aptitudes técnicas dentro del proceso de reclutamiento para Coppel - Sistemas Diseño.

La aplicación esta desarrollada en PHP atraves de Laravel 10.8.
El back end esta implementado en api rest mientras que el front en html con javascrip.
